# Smart Contact Manager

A full-stack contact management application that goes beyond basic CRUD by integrating real-world **Data Structures & Algorithms** for performance and scalability.

## 🚀 Features

* Add, update, and delete contacts
* Fast prefix search using **Trie**
* Optimized data fetching with **LRU Cache**
* Contact relationship mapping using **Graph (BFS/DFS)**
* Interactive UI with modern glassmorphism design

## 🛠 Tech Stack

* **Backend:** Node.js, Express
* **Database:** MongoDB
* **Frontend:** HTML, CSS, JavaScript
* **Core Concepts:** Trie, LRU Cache, Graph

## ⚙️ Setup

```bash
git clone https://github.com/your-username/smart-contact-manager.git
cd smart-contact-manager/backend
npm install
npm start
```

Open in browser:

```
http://localhost:5000
```

## 📌 Highlights

* Efficient search in **O(m)** using Trie
* Reduced database calls via caching
* Graph traversal for real-world contact networks
